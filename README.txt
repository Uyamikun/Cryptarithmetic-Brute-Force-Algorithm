Deskripsi :
Program untuk mencari solusi cryptarithmetic dengan Brute Foce

1. Program menjalankan hanya dari file
2. jika ingin membuat file test tolong gunakan tanda (-) minus untuk garisnya dan tanda (+) berada di operand sebelah kanan (lihat contoh)
3. gunakan spasi untuk memisahkan antara satuan,puluhan,ratusan,...

CONTOH
  SOAL+

Cara menggunakan program

Linux :
1. Extract Tucil1_13519168.zip
2. Copy file test case yang ingin dibuka ke folder yang berisi main.py
3. Buka terminal dan masuk ke path folder yang berisi main.py
4. Tulis "python3 main.py" tanpa tanda kutip

Windows :
1. Extract Tucil1_13519168.zip
2. Copy file test case yang ingin dibuka ke folder bin
3. Buka "main.exe" di folder bin
